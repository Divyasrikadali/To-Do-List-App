A vibrant, interactive, and user-friendly To-Do web application that lets you manage tasks efficiently with features like pinning, due dates, progress tracking, and theme toggling. Perfect for keeping your tasks organized and visually appealing.

**Features**

Add Tasks: Easily add tasks with optional due dates.

Delete Tasks: Remove tasks instantly.

Pin Tasks: Keep important tasks at the top.

Mark as Completed: Click on a task to mark it complete; completion updates progress.

Overdue Highlighting: Tasks past their due date are highlighted.

Progress Tracking: Visual progress bar showing percentage of completed tasks.

Search & Filter: Search tasks by text and filter by All, Active, or Completed.

Dark & Light Themes: Toggle between vibrant light and dark modes.

Drag & Drop: Rearrange tasks using drag-and-drop.

Task Reminders: Browser notifications for tasks with due dates.
